
# docker-Pyfreebilling

Run Pyfreebilling in a Docker container

## Usage

### Clone
   
    $ git clone https://github.com/droidfanat/droidfanat-pyfreebilling_17_docker.git ./pyfreebilling
    
### Build
    $cd ./pyfreebilling
    $ docker-compose build .
### Run
Foreground:
    $ docker-compose up -d
